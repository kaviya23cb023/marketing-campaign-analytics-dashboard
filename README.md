# 📊 Marketing Campaign Performance Analytics Dashboard

## 🚀 Project Overview

This project is an AI-powered marketing analytics dashboard that helps businesses analyze campaign performance, predict customer behavior, and make data-driven decisions.

---

## 🎯 Features

* 📊 Business Dashboard with KPIs (Revenue, ROI, Conversion Rate)
* 🤖 Machine Learning Models (Logistic Regression, Random Forest)
* 📈 Model Evaluation (Accuracy, Precision, Recall, Confusion Matrix)
* 🔮 Real-Time Customer Purchase Prediction
* 💡 Automated Business Insights
* 🎨 Interactive UI with animations using Streamlit

---

## 🧠 Technologies Used

* Python
* Streamlit
* Scikit-learn
* Pandas, NumPy
* Matplotlib, Seaborn

---

## 📂 Dataset

* Synthetic dataset with 5000 customer records
* Includes demographics, campaign details, and purchase behavior

---

## ▶️ How to Run

```bash
pip install -r requirements.txt
python -m streamlit run app.py
```

---

## 🌐 Deployment

This project can be deployed using Streamlit Cloud.

---

## 📌 Key Insights

* Campaign performance varies across channels
* Customer income and ad interaction strongly influence purchases
* Random Forest model provides better prediction accuracy
